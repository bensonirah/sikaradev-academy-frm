# Without for loop

```javascript
text += cars[0] + '<br>'
text += cars[1] + '<br>'
text += cars[2] + '<br>'
text += cars[3] + '<br>'
text += cars[4] + '<br>'
text += cars[5] + '<br>'
```

# With for loop

```javascript
for (let i = 0; i < cars.length; i++) {
  text += cars[i] + '<br>'
}
```
